# Mail::X::Smtpapi

Integrates a custom X-SMTPAPI field into the [Mail](https://github.com/mikel/mail) gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mail-x_smtpapi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mail-x_smtpapi
